ol 태그 : 순서가 존재하는 목록 태그

li 태그 : 목록의 한 줄(목록 중 낱개 1개)

        type : 표시되는 숫자 모양을 변경
        type="A" : 영어 대문자로 시작(A, B, C, D, ...)
        type="a" : 영어 소문자로 시작(a, b, c, d, ...)
        type="i" : 로마 소문자로 시작(i, ii, iii, iv, ...)
        type="I" : 로마 대문자로 시작(I, II, III, IV, ...)
    교실에 존재하는 물건들
  1. 컴퓨터
  2. 노트북
  3. 의자
  4. 조명

dl 태그 : 설명이나 목록을 작성하는 태그

dt 태그 : 설명 제목을 작성하는 태그
