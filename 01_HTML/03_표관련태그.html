표 관련 태그

title(표) 태그

        - 웹 문서에서 자료(데이터) 정리를 위해 사용하는 태그
        - table 태그는 행, 열을 작성하기 위한 판을 나타냄

        tr : 테이블에서 가로 한 줄을 나타내는 태그
            table row
        th : 두꺼운 글씨로 표현, 가운데 정렬이 자동으로 적용
            table head
        td : 테이블에서 가로 한 줄에서 한 칸을 나타내는 태그
            table data
