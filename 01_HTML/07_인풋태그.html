form / input 태그

form 태그

        - 입력 양식을 작성하는 공간
        - 내부에 작성된 입력 값을 지정된 곳으로 제출하는 역할

        [속성]
        1) action : 제출될 곳을 지정하는 속성
        2) method : 설정된 곳으로 데이터를 전달하는 방식
        3) name   : 이름을 지정하는 속성 / 지정된 이름은 DB에 전달하는 이름값이됨
        4) target : 제출된 페이지를 현재 / 새 탭으로 열기 선택해서 설정

input 태그

        - 웹 문서에서 사용자(클라이언트)가 입력할 수 있는
        양식을 제공하는태그

        - 많은 type 존재하고, input의 변경 형태도 다수 존재

        * input 태그에는 name 속성이 필수로 작성되어야 한다.
        (꼭은 아니지만 작성하는 경우 많음 80% 이상)

숫자 관련 input 태그

        type="number" : 숫자만 입력 가능한 입력 상자
        type="range"  : 슬라이드 바를 이용해서 숫자를 지정
                        (단독 사용 불가, JavaScript 와 함께 사용 가능)

        [속성]
        1. min : 최소값
        2. max : 최대값
        3. step : 숫자가 증가/ 감소하는 단계 (간격)
        4. value : 초기 값(기본 값) 지정

날짜 / 시간 관련 input 태그

radio, checkbox

        - 여러 연관된 값을 묶어서 선택하는 경우 사용
        - radio : 여러 값 중 하나 선택( ex: 성별)
        - checkbox : 여러값 중 원하는 것을 하나, 두 가지 이상, 하나도 선택안함, 모두 선택
                    처럼 선택의 범위를 다양하게 선택할 수 있음
                    (ex: 상세검색, 관심분야, 취미)

radio 확인

A팀 B팀 C팀 D팀

checkbox 확인

red yellow orange green

select / option 태그

        - select : 클릭하면밑으로 메뉴가 내려오는 입력 태그
        - option : select 클릭 시 보여질 메뉴를 작성하는 태그
