img 태그

        - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그
        [속성]
        시작 태그 다음에 띄어쓰기로 구분해서 작성하는 내용
        1.src : 삽입할 이미지의 경로를 작성하는 속성
                (파일 위치, 인터넷 주소)
        2. width / height : 가로 크기 / 세로 크기를 지정하는 속성
        3. alt : 이미지 설명을 작성하는 속성
                    이미지가 출력되지 않을 경우 화면에 이미지 명칭 보여줌
                    + 스크린 리더가 이미지의 설명을 읽어줌
